# AfriFoundry Data Collector

Professional field data collection tool for African innovation intelligence.

## Features
- Offline-first architecture
- Field data & survey modes
- GPS location capture
- Photo storage
- Export to JSON/CSV
- Progressive Web App (installable)

## Development
```bash
npm install
npm run dev
```

## Deployment
Deployed on Vercel: https://collector.afrifoundry.ai
